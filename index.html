QuickShip Customer Profile

This repository stores the source code. The working data-entry app must be deployed from Google Apps Script.

GitHub Pages can show static files only. It cannot run google.script.run, save to Google Sheets, or process Apps Script template includes.

Publish the live app

  1. Open the project in Google Apps Script.
  2. Run setupQuickShipSheet() once to create the data sheet.
  3. Click Deploy, then New deployment.
  4. Select Web app.
  5. Set Execute as to Me.
  6. Set access as needed, then copy the Apps Script web app URL.

Use that Apps Script web app URL as the real QuickShip form link.
